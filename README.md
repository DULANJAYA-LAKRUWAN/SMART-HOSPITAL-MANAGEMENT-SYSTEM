# SMART HOSPITAL MANAGEMENT SYSTEM

## Project Overview
The Smart Hospital Management System is a comprehensive application designed to streamline and enhance the operations of healthcare facilities, ensuring efficient management of patient data, appointments, and medical records.

## Features
- Patient Registration and Management
- Appointment Scheduling
- Medical Records Management
- Billing and Payment Processing
- User Authentication and Role-Based Access Control

## Technology Stack
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: Node.js, Express
- Database: MongoDB

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/DULANJAYA-LAKRUWAN/SMART-HOSPITAL-MANAGEMENT-SYSTEM.git
   ```
2. Navigate to the project directory:
   ```bash
   cd SMART-HOSPITAL-MANAGEMENT-SYSTEM
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Database Setup
1. Install MongoDB on your local machine or create a MongoDB Atlas account.
2. Create a database named `hospital_management`.
3. Update the connection string in your environment variables.

## Usage Guide
To run the application:
```bash
npm start
```
Navigate to `http://localhost:3000` in your browser to access the application.
